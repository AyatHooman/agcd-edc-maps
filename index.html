AGCD EDC — Fortran SI vs Python OI

Monthly comparison maps, 2025. Interactive 6-panel comparison of the operational Fortran statistical interpolation and the Python optimal-interpolation engine. Top row — station maps: observed value, Fortran (obs − grid), Python (obs − grid). Bottom row — Fortran rainfall, Python rainfall, and their difference. Hover a gauge for its values; scroll to zoom (all panels zoom together).

Three station-selection variants:
No filter — every real gauge fed straight to Python, no extra screening (all markers grey).
Fortran filter — Python with the replicated Fortran cross-validation (xvaldt) screening (PASS / REJECT).
Python filter — Python with the new station-QC class (boundary, climatology and spatial-neighbour tests): PASS / SUSPICIOUS / REJECT.

Every gauge is shown, including the ones a filter rejects. On the bottom row the station markers are coloured by QC status — ● PASS, ● SUSPICIOUS, ● REJECT, ● none — and every hover popup reports the gauge's QC status and flags.
