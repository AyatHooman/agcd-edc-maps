AGCD EDC — Fortran SI vs Python OI

Monthly comparison maps, 2025. Interactive 6-panel comparison of the operational Fortran statistical interpolation and the Python optimal-interpolation engine. Top row — station maps: observed value, Fortran (obs − grid), Python (obs − grid). Bottom row — Fortran rainfall, Python rainfall, and their difference. Hover a gauge for its values; scroll to zoom (all panels zoom together).

Two station-selection variants:
No filter — every Fortran xdataobs station + pseudo-obs fed straight to Python (no extra screening).
Fortran filter — Python with the replicated Fortran cross-validation (xvaldt) station screening applied.
